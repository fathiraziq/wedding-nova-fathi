# Wedding Invitation Image Assets

All production photos live inside this folder. Source/reference photos should not be kept in the project root after they have been cropped and exported.

## Folder Structure

```
assets/images/
|-- cover/       -> opening invitation background
|-- hero/        -> main couple photo card
|-- couple/      -> bride and groom profile photos
|-- gallery/     -> gallery carousel photos
`-- README.md    -> this file
```

## Current Files

| Area | File | Size |
| --- | --- | --- |
| Cover | `cover/cover.jpg` | 900 x 1200 |
| Hero | `hero/couple.jpg` | 1084 x 1354 |
| Bride | `couple/bride.jpeg` | 1134 x 1418 |
| Groom | `couple/groom.jpeg` | 1134 x 1418 |
| Gallery | `gallery/1.jpg` to `gallery/16.jpg` | 3:4 portrait assets, alternating adat/jas |

## Notes

- Keep image dimensions aligned with the existing design slots.
- Keep optimized assets in `assets/images/...`.
- Remove temporary source/reference photos from the project root before committing.
