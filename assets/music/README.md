# Background Music

Letakkan file musik background di folder ini.

## Format yang didukung
- `.mp3` (recommended, kompatibel semua browser)
- `.ogg` (fallback)

## Cara pakai
1. Rename file musik menjadi `bg-music.mp3`
2. Letakkan di folder ini: `assets/music/bg-music.mp3`

## Rekomendasi
- Durasi: 2-4 menit (akan di-loop otomatis)
- File size: < 3MB (untuk loading cepat)
- Genre: instrumental, acoustic, atau nasheed lembut
- Volume: normalize ke -14 LUFS agar tidak terlalu keras
